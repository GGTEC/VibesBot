VibesBot Autenticação

Olá, seja bem-vindo ao VibesBot

Para iniciarmos o aplicativo é necessário efetuar a autenticação da sua conta, a autenticação é necessária para termos acessos á alguns dados importantes para o funcionamento correto do sistema, fique tranquilo, todos os dados de autenticação são salvos localmente e somente você e o TikTok tem acesso ao token de autenticação.

Na autenticação será necessário informar o seu SESSIONID e o SIDGUARD, essas informações são para o aplicativo conseguir se conectar totalmente ao tiktok, Caso opte por não inserir estas informações as mensagens de certos usuários não irão aparecer no chat.

Bom, agora que já esclarecemos algumas coisas, clique no botão abaixo para iniciarmos.
